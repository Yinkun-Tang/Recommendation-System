# 推荐系统平台

推荐系统是机器学习的重要分支和应用方向，也是计算机科学领域的研究热点。本项目旨在模拟视频平台的浏览推荐机制，同时为研究者和开发者提供探索不同推荐策略与方法性能的实验平台。

本项目实现了四种主流推荐范式，每种均封装为模块化类结构：
1. 用户协同过滤
2. 物品协同过滤  
3. 内容推荐
4. 混合推荐

针对冷启动场景，默认使用基于流行度的推荐器。

系统支持独立的评估流程，可通过不同的训练/测试矩阵和指标进行离线实验。

## 项目结构
- `backend/` : Python后端，包含数据预处理、推荐算法、评估模块
- `frontend/` : 基于Node.js构建的前端界面
- `data/` : 原始与处理后数据集存储
- `documents/` : 供参考的补充材料

## 环境要求

1. 安装最新版本Node.js
2. 创建Python虚拟环境，按 ```requirements.txt``` 安装所有依赖包
3. 在 ```./frontend``` 目录下运行 ```npm install``` 安装前端依赖
4. 从 [MovieLen | GroupLens](https://grouplens.org/datasets/movielens/) 下载MovieLens 1M数据集
5. 解压下载的数据集，将 ```ml-1m``` 文件夹提取到 ```./data/raw``` 目录下

## 安装步骤

1. 克隆或下载本仓库

## 使用方法
*注：以下步骤在WSL环境（已安装Chrome浏览器）下验证通过。详细实现请查看相关shell脚本文件。*

### Streamlit评估模式
*注：此模式用于模型评估。如无需评估，可忽略第一步，且无需执行第二步中的 ```evaluation_data_preprocessing.py```*
1. 如需进行模型评估，可在 ```evaluation_config.json``` 中调整参数选择、预处理或计算策略
2. 确认配置后，在 ```./backend``` 目录下运行 ```data_preprocessing.py``` 和 ```evaluation_data_preprocessing.py```
3. 数据预处理完成后，在根目录运行 ```start_st.sh```，并按界面提示操作

### React前端 + FastAPI后端服务器
*注：此模式不包含评估功能，每次会随机选择用户ID进行推荐。可通过刷新页面随机选择新用户ID*
1. 开始前，请在 ```./backend``` 目录下运行 ```data_preprocessing.py```
2. 在根目录运行 ```start_dev.sh``` 启动前端界面

## 项目特点

### 🎯 核心功能
- **多算法对比**：支持协同过滤、内容推荐、混合推荐等多种算法
- **冷启动处理**：内置基于流行度的冷启动推荐机制
- **完整评估流程**：提供离线评估与可视化结果展示
- **模块化设计**：各推荐器独立封装，便于扩展

### 🔧 技术栈
- **后端**：Python, FastAPI, Scikit-learn, Pandas
- **前端**：Node.js, React, Streamlit
- **评估工具**：多种推荐系统指标（准确率、召回率、覆盖率等）
